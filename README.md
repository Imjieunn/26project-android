#### Description
* 성향 테스트 안드로이드 어플리케이션
* 언어: java
* IDE: android studio

#### Process
1. 메인 화면 구현,
2. 첫 번째 게임 구현 (카드 뒤집기 게임)
3. 메인 화면과 연결, 메인 화면 애니메이션
4. 게임 기록 저장 구현
5. 두 번째 게임 구현 (벽돌깨기 게임)
6. 메인 화면과 연결, 애니메이션 
7. 세 번째 게임 구현, 메인 화면과 연결, 애니메이션 (계단오르기 게임)
8. 프로젝트 마무리
